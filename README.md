# Mẫu Test Case Web Form

Cấu trúc thư mục:

```text
testcase_web_project_v5/
  index.html
  assets/
    css/
      style.css
    js/
      app.js
```

## Cách dùng

1. Mở `index.html` bằng trình duyệt.
2. Nhập nội dung test case trực tiếp trên form.
3. Dùng nút `+` / `−` để thêm hoặc xóa dòng.
4. Chọn `Status` và `Severity` bằng dropdown.
5. Bấm `In / Lưu PDF` để in hoặc lưu thành PDF.

## Ghi chú

- CSS nằm tại `assets/css/style.css`.
- JavaScript nằm tại `assets/js/app.js`.
- PDF tạo bằng chức năng in của trình duyệt là PDF tĩnh. Muốn PDF điền được form cần tạo riêng dạng Fillable PDF / AcroForm.
